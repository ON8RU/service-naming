# service-naming

Istallation

```bash
sudo wget https://raw.githubusercontent.com/ON8RU/service-naming/refs/heads/master/genn.sh -O /usr/bin/genn
chmod +x /usr/bin/genn

```

## Usage

Default names

```bash
genn
```

Slugified names

```bash
genn -slugify
```

### Animals

```bash
genn
```

or

```bash
genn -of animals
```

### Space

```bash
genn -of space
```
